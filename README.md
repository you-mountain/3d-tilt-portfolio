## 🚀 Getting Started

### 1. Clone or download
```bash
git clone https://github.com/your-username/3d-tilt-portfolio.git
cd 3d-tilt-portfolio

```

2. Open index.html in your browser
Double‑click the file or use Live Server in VS Code.

3. Customize content
Edit these sections inside index.html:

Section	Line range (approx)	What to change
Name & title	207–210	Elias Voss, your job title
Bio	211–213	Your personal description
Skills	246–256	Tech stack & proficiency level
Work Experience	260–270	Your job history
Projects	280–298	Portfolio items & links
Social links	346–352	Your profile URLs
Contact form	325–337	(No backend — add Formspree/etc)
🎨 Customization Tips
🔄 Change avatar icon
Replace line ~190:

```bash
html
<i class="fa-solid fa-chess-queen"></i>
with your own <img src="photo.jpg"> inside .avatar-frame.

🎨 Adjust color palette
Edit CSS variables (lines 18–27):
css
--card-bg: #111318;    /* card background */
--gold: #c8a870;       /* accent color */
--text-light: #e8e0d0; /* main text */
```

🖼️ Add real project images
Replace the icon with <img> inside .project-card or keep icons for minimal style.
